# Valentine's Day Message Website

A beautiful, animated Valentine's Day message website with music and interactive elements.

## Features
- Beautiful landing page with entrance button
- Animated message page with rotating images
- Background love song with player controls
- Floating heart animations
- Fully responsive design

## How to Use
1. Download all three files (index.html, style.css, script.js)
2. Place them in the same folder
3. Open index.html in your web browser
4. Customize the message and name in script.js

## Customization
1. Change the name in script.js (CONFIG.yourName)
2. Replace the image URLs in script.js (CONFIG.images)
3. Change the music by updating the audio source in index.html
4. Modify colors in style.css

## Browser Compatibility
Works on all modern browsers (Chrome, Firefox, Safari, Edge)
